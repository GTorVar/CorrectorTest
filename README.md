# CorrectorTest

La idea de este programa es servir de plantilla correctora del test SCL90 para población española. El baremo está obtenido de un estudio con población general de González Sanguino et al. (2007). 

El programa está compilado para funcionar en Windows 64 bits, aunque desarrollado en Ubuntu (si da problemas de compatibilidad agradezco el aviso). Es simple, son 3 columnas para introducir manualmente las respuestas del SCL90R de la persona evaluada, es necesario tanto introducir el nombre del mismo como del evaluador; es necesario introducir el sexo de la persona evaluada para tener en cuenta la diferencia en los baremos. La fecha se puede cambiar aunque de manera predeterminada coge la del día en curso.

Al pulsar el botón "Corregir", se genera un PDF con la hoja de respuestas de la persona evaluada y la escala con los posibles problemas presentados.

IMPORTANTE: Este programa no guarda ni almacena ningún dato de las personas evaluadas, al salir del mismo no queda registro de las respuestas. Los PDF con la información se deben ceñir a lo dispuesto en la legislación de cada país sobre el uso de datos de caracter sanitario.
